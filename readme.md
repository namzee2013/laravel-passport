## Laravel JWT


## How to setup:

  1.  clone repo
      ```
      git clone https://github.com/namzee2013/laravel-passport.git
      ```

  2. install packages

     ```
     composer install
     ```

  3. create and update .env

  4. generate key

     ```
     php artisan key:generate
     ```

  5. migrate database

     ```
     php artisan migrate
     ```

  6. install passport

     ```
     php artisan passport:install
     ```

  7. run project

     ```
     php artisan serve
     ```
